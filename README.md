# homebrew-tap

Homebrew formulae for [CMD+RVL](https://github.com/cmdrvl) tools.

## Install

```bash
brew tap cmdrvl/tap
brew install cmdrvl/tap/airlock
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [airlock](https://github.com/cmdrvl/airlock) | Prove what crossed the model boundary |
| [assess](https://github.com/cmdrvl/assess) | Deterministic decision classification over a spine evidence bundle |
| [canon](https://github.com/cmdrvl/canon) | Resolve messy identifiers to canonical IDs using versioned registries |
| [cmdrvl-benchmark](https://github.com/cmdrvl/benchmark) | Gold-set scoring for extraction pipelines |
| [cmdrvl-hash](https://github.com/cmdrvl/hash) | Streaming content hashing for JSONL manifest records |
| [fingerprint](https://github.com/cmdrvl/fingerprint) | Determine whether an artifact matches a known template |
| [lock](https://github.com/cmdrvl/lock) | Pin artifacts, fingerprints, and tool versions into a self-hashed dataset lockfile |
| [pack](https://github.com/cmdrvl/pack) | Seal artifacts into immutable, self-verifiable evidence packs |
| [profile](https://github.com/cmdrvl/profile) | Creates, validates, and freezes column-scoping profiles for report tools |
| [rvl](https://github.com/cmdrvl/rvl) | Reveal the smallest set of numeric changes that explain what actually changed |
| [shape](https://github.com/cmdrvl/shape) | Structural comparability gate for CSV datasets |
| [vacuum](https://github.com/cmdrvl/vacuum) | Inventory artifacts into deterministic JSONL manifests |
| [verify](https://github.com/cmdrvl/verify) | Deterministic constraint evaluation for the epistemic spine |
